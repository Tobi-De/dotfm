Colophon: what is the machinery behind this?

I built this site with django (completely overkill I know), tailwind css and htmx. The blog was built using coltrane, a django based content site framework. The app is deployed on a linode vps and managed with caprover.

I didn't design this myself, here is a non-exhaustive list of projects I was inspired by:

  • A template by the creators of Tailwind CSS
  • Lee Robinson personal website
  • Josh W Comeau blog
  • Adam Johnson personal website
The source code of this app is available on github.

This is my second attempt at this, the previous version was also built with django and the source code is also available on github.

There are probably thousands of better ways to build this and I would suggest pure HTML, CSS and javascript for something this simple or a static site generator like Pelican
Thanks to all of these technologies for making my blog possible!
